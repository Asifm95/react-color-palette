## Installation

    $ git clone https://github.com/Asifm95/react-color-palette.git
    $ cd react-color-palette
    $ npm install

## Start & watch

    $ npm start

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Simple build for production

    $ npm run build
